Install Guide
----------

```sh
curl https://install.meteor.com/ | sh
git clone https://github.com/HedCET/Torrent-Alert.git Torrent-Alert
cd Torrent-Alert
export TORRENTZ_URL="https://torrentz2.eu" && meteor
```
